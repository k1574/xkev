xkev
====

simply simulate KeyPress/KeyRelease

Installation
------------

	% make
	% make install

This will create by default `xkev`

Usage
-----

	xkev -e XF86_Back
	xkev -e XF86_Forward
	xkev -e XF86_AudioMute

send Control-c

	xkev -c -e c 

send Control-Alt-Del
	
	xkev -c -a -e Delete
